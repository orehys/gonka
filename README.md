# Gonka

Полнофункциональный Django-проект интернет-магазина с кастомной админ-панелью, системой синхронизации поставщиков, SEO-контуром и автоматизацией через systemd.

Проект ориентирован на реальную эксплуатацию: большой каталог, регулярные импорты, контроль цен и остатков, менеджерская админка без зависимости от стандартного Django Admin.

---

## Основные возможности

### Каталог и витрина
- Иерархический каталог (категория → подкатегория)
- Поиск по:
  - названию
  - SKU (с нормализацией)
  - описанию
  - категориям и характеристикам
- Фильтрация по:
  - характеристикам
  - брендам
  - маркам / моделям авто
  - наличию
- Сортировка по цене и новизне
- Пагинация с сохранением фильтров
- Карточка товара:
  - галерея изображений
  - характеристики
  - хлебные крошки
  - избранное
  - недавно просмотренные

---

### Корзина и заказы
- Корзина на сессиях (без сторонних cookies)
- AJAX-добавление и обновление количества
- Подсчёт:
  - общей суммы
  - скидки (old_price → price)
- Оформление заказа без регистрации
- История заказов в личном кабинете
- Проверка остатков перед оформлением

---

## Синхронизация и автоматизация

Проект использует **Django management-команды**.

### Контроль цен
- Автоматическое скрытие товаров с ценой `0` или `NULL`
- Автоматическое возвращение товаров при появлении цены
- Батч-обновления


---

## Кастомная админка (Backoffice)

Админка полностью адаптирована под бизнес-логику проекта.

### Товары
- Список товаров с фильтрами
- Ручное создание товара:
  - SKU, цена, себестоимость
  - категории
  - характеристики текстом
  - массовая загрузка изображений
- Управление видимостью
- Редактирование характеристик и изображений

### Заказы
- Поиск по ID / email / телефону
- Фильтр по статусу
- Inline-смена статуса заказа
- Просмотр состава заказа, доставки и комментариев

### Пользователи
- Поиск по логину / email / ФИО / телефону
- Статус аккаунта
- История регистрации и входов

---

## Чип-тюнинг
- Каталог конфигураций:
  - марка
  - модель
  - год
  - двигатель
- Несколько Stage (Stage 1 / Stage 2 / …)
- Для каждого Stage:
  - набор параметров (метрик)
  - заводские / тюнинг-значения
  - дельта
  - цена
- Inline-редактирование без вложенных форм
- Заявки на чип-тюнинг с управлением статусами

## Установка и запуск (dev)

```bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### Переменные окружения

Создайте `.env`:

```env
DEBUG=1
SECRET_KEY=change-me
ALLOWED_HOSTS=localhost,127.0.0.1
```

### Миграции и запуск

```bash
python manage.py migrate
python manage.py runserver
```
